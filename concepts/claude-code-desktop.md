# Claude Code Desktop

> Claude Code 桌面端应用。2026-04-15 完成界面重设计，核心变化是支持在同一个窗口里并排运行多个 Claude 会话，并集成一套开发工具面板。

## 核心能力

- **多会话侧边栏**：同一个窗口里并排运行多个 Claude 会话，左侧侧边栏统一管理，支持按状态、项目或环境过滤
- **集成开发工具**：内置终端、文件编辑器、Diff 查看器（为大变更集优化）、HTML 预览、PDF 预览
- **面板自由布局**：所有工具面板支持拖拽排列
- **Side Chat**：快捷键 ⌘+;（Mac）/ Ctrl+;（Windows），在不打断主任务的情况下临时问 Claude 一个问题，结合当前上下文回答但不影响主线
- **CLI 对齐**：CLI 插件原本支持的功能桌面端全部对齐，完全一致
- **Mac SSH 支持**：SSH 支持扩展到 Mac（此前仅 Linux）

## 设计目标：单窗口管全局

改版前的痛点是并行处理多个任务只能开多个终端窗口或切桌面，切换很繁琐。重设计后所有会话收进侧边栏，点击切换或直接在多个会话面板之间操作，不需要来回跳窗口。

## 与 Routines 的关系

同日发布的 [[concepts/claude-code-routines]] 让 AI 在云端异步执行任务，桌面端让用户在本地同步管理多个会话。方向一致——都在往「并行」走。Anthropic 的思路是把 Claude Code 从编程助手做成能接管重复工程工作、在后台持续运转的自动化平台。
